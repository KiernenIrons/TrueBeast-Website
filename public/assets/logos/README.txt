LOGOS FOLDER
============
Drop your logo files here. The website references these files:

  logo.png         - Main TrueBeast logo (used in navbar and footer)
  logo-dark.png    - Dark/alternative version (optional)
  favicon.ico      - Browser tab icon (also copy this to the root folder)

Recommended sizes:
  logo.png   → 200x60px (or similar wide format, transparent background)
  favicon    → 32x32px or 64x64px
