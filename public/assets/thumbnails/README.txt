THUMBNAILS FOLDER
=================
OPTIONAL: You can drop custom video thumbnail images here.

However, the website is already set up to automatically load thumbnails
directly from YouTube using your video IDs set in js/config.js.

You only need to add files here if you want to override a YouTube thumbnail
with a custom image. If so, name them:

  video-1.jpg
  video-2.jpg
  video-3.jpg
  video-4.jpg

Recommended size: 1280x720px (16:9)

To use these instead of YouTube thumbnails, update the video entries
in js/config.js by adding: customThumbnail: 'assets/thumbnails/video-1.jpg'
